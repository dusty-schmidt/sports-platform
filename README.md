## SABERSIM AS THE INSPIRATION
- play by play simulators generate realistic projections
- simulations done by slate, and saved as full slate sims.
- optimizer uses buckets of full slate sims, averages projections, builds a lineup.
- each lineup is from a unique bucket
- reducing bucket size simulates higher variance realistically




## DIRECTORY CONTENTS
- 'tennis-sims' - fastapi template build

- 'dfs-pools' - fully working draftkings player pool provider
